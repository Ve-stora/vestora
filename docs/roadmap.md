# Vestora — Expansion Roadmap

## Phase 1: NSE (Now – End 2026)

**Objective**: Build depth on the Nairobi Securities Exchange. Acquire first B2B clients and retail users.

### Milestones
- [ ] Real-time NSE equity data pipeline (live scraping + normalization)
- [ ] XGBoost price forecasting models trained on NSE historical data
- [ ] Isolation Forest anomaly detection on NSE volume/price
- [ ] B2B API v1 — structured endpoints for broker integration
- [ ] First paying B2B client (target: Kenyan broker or SACCO)
- [ ] 500 active retail users
- [ ] v1.0.0 production release
- [ ] NSE bond and T-bill data integration

### Target Partnerships
- Nairobi-based licensed brokers (Dyer & Blair, African Alliance Kenya)
- SACCOs with investment mandates
- Kenyan fintech platforms seeking analytics layer



## Phase 2: USE (Early 2027)

**Objective**: Add Uganda Securities Exchange as secondary market. Leverage Ugandan founder credibility and diaspora angle.

### Milestones
- [ ] USE equity data pipeline
- [ ] BOU government securities (T-bills, bonds) integration
- [ ] Diaspora-facing product features (FX context, cross-border portfolio view)
- [ ] Partnership with USE-licensed broker (Crested Capital, African Alliance Uganda)
- [ ] URSB brand protection registration for Vestora

### Why USE Now and Not Earlier
USE has 17 listed companies and thin daily liquidity — insufficient for ML signal generation at launch. By Phase 2, Vestora's models will be mature enough to handle thin-market conditions, and the Ugandan institutional market (pension funds, SACCOs) provides a B2B revenue path independent of retail trading volume.



## Phase 3: DSE + RSE (Mid 2027)

**Objective**: Expand to Tanzania and Rwanda — both growing markets with improving data infrastructure.

### Milestones
- [ ] DSE (Dar es Salaam Stock Exchange) data pipeline
- [ ] RSE (Rwanda Stock Exchange) data pipeline
- [ ] Unified EAC market dashboard
- [ ] Cross-market correlation and arbitrage signal detection
- [ ] East African institutional client acquisition (pension funds, development banks)



## Phase 4: Pan-African (2028+)

**Objective**: Scale to South Africa, North Africa, West Africa and beyond. Position Vestora as the Bloomberg Intelligence equivalent for African capital markets.

### Target Exchanges
- GSE (Ghana Stock Exchange)
- NGX (Nigerian Exchange Group)
- JSE (Johannesburg Stock Exchange) — aspirational
- BRVM (West African regional exchange)

### Funding Target
Series A from impact-aligned VCs and development finance institutions: FSD Africa, UNCDF, Ventures Platform, Catalyst Fund, Helios Investment Partners.
